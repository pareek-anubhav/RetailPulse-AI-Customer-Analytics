# RetailPulse AI - Customer Analytics & Demand Forecasting Dashboard

## Project Overview

RetailPulse AI is an end-to-end retail analytics platform developed as part of a Data Science & Analytics Internship.

The project helps businesses analyze sales performance, understand customer behavior, predict future demand, identify potential customer churn, and optimize inventory management through interactive dashboards and machine learning models.

The solution combines data analytics, forecasting, and business intelligence into a single user-friendly Streamlit application.

## Dashboard Pages

### 1. Sales Dashboard
- Revenue and sales KPIs
- Monthly sales trends
- Country-wise sales analysis
- Product performance insights
- Interactive filters

### 2. Customer Dashboard
- Customer segmentation analysis
- RFM-based customer categorization
- Customer distribution visualization
- Churn customer insights
- Segment-wise revenue contribution

### 3. Forecast Dashboard
- Demand forecasting using machine learning
- Sales trend prediction
- Forecast visualization
- Future demand analysis
- Forecast performance monitoring

### 4. Inventory Dashboard
- Inventory optimization insights
- Reorder recommendations
- Safety stock monitoring
- Demand-based inventory planning
- Stock management analytics
